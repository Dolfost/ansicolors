description:
Change the color and mode of the text by passing constants to the stream
by your favorite method of output such as 'cout' or 'printf()', or using
functions setBKGRND() and setFRGRND().
(see list below)

usage:
void setFOREGROUND(const short code), void setFRGRND(const short code)

void return : 		has no return value,
setFRGRND : 		set foreground (text) color,
const short code :	color value [0 - 255];


void setBACKGROUND(const *char), void setBKGRND(const *char)

void return : 		has no return value,
setBACKGROUND :		set background color,
const char *code :	color value [0 - 255], reads only code[0][1][2] values;

changes:
2.0-relase/rework1 28/07/2021		fully reworked the libriary, added the 
					setFRGRND() and setBKGRND() functions;

syntax: 
XXX   - foreground color,
mXXX  - text mode (style),
bXXX  - background color,
xXXXh - high intensty color;

example: 
#include <iostream>
#include "ansicolors/ansicolors.hpp"
int main(void) {
	std::cout << RED ITALIC "Hello!" sRST "\n"; // red itaic text
	std::cout << BLK bWHT BLD "Im an colorful libriary!" sRST "\n"; // black bold text on white background
	return 0;
}
