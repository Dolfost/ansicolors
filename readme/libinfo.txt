description:
Change the color and mode of the text by passing constants to the stream
by your favorite method of output such as 'cout' or 'printf()', or using
functions setBKGRND() and setFRGRND().
(see list below)

usage:
     setFRGRND                            setFRGRND
void setFOREGROUND(const short arg), void setFOREGROUND(const *char code)

void return : 		has no return value,
setFOREGROUND : 	set foreground (text) color,
setFRGRND :		alias (you can call a function through it),
const char *code &
const short arg :	color value [0 - 255];

     setBKGRND				   setBKGRND
void setBACKGROUND(const short arg), void setBACKGROUND(const *char code)

void return : 		has no return value,
setBACKGROUND :		set background color,
setBKGRND : 		alias (you can call a function through it),
const char *code &
const short arg:	color value in char string [0 - 255];

tips:
Functions above read no more than the first three elements of the
array or the first three digits of the number passed to the function.

changes:

				than 4 bytes is passed to the functions;

syntax: 
XXX   - foreground color,
mXXX  - text mode (style),
bXXX  - background color,
xXXXh - high intensty color;

example: 
#include <iostream>
#include "ansicolors/ansicolors.hpp"
int main(void) {
	std::cout << RED ITALIC "Hello!" sRST "\n"; // red itaic text

	std::cout << BLK bWHT BLD "Im an colorful libriary!" sRST "\n";
	// black bold text on white background

	return 0;
}
